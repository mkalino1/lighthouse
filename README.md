# Lighthouse game
Godot 2D game about guiding ships and fighting monsters.

![Gameplay](/gameplay.png)

## Gameplay
- Use the arrows/WASD keys to move your character.
- Use the mouse to fire bullets at the monsters that will try to attack you, ships and lighthouse.
- The more ships you protect during their journey, the more points you earn.
- Avoid the monsters and obstacles in the water.
- Collect power-ups to gain special abilities and build better lighthouse. More powerful lighthouse can guide ships faster.
